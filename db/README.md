# /db

Database schemas, migrations, and seed data.

Houses Prisma schema files, migration history, seed scripts, and any
database-related utilities. The Prisma schema is currently at `prisma/schema.prisma`.

## Contents (planned)
- `migrations/` — Prisma migration history
- `seeds/` — Seed scripts for development and staging
- `scripts/` — One-off data migration or maintenance scripts
